## 特别提示
要正常运行该app，必须保持mongodb运行状态
我是mac用户 大致说下 下载好对应的文件后，在根目录（是/目录，不是~目录）
下创建 data/db的文件 并且授权读写sudo chrown 用户名 /data/db，
then 在安装的mongodb目录下的bin运行 ./mongod  
last 新开窗口仍然在bin下运行 ./mongo
就可以开开心心地运行我们地程序啦
